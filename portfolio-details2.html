Quiz Combate a Dengue

Ao desenvolver o quiz sobre a dengue, aprendi a estruturar dados em JavaScript com arrays e objetos para organizar perguntas e respostas. Manipulei o DOM para gerar elementos HTML dinamicamente e implementei lógica para verificar respostas e calcular acertos em tempo real. Utilizei o método `cloneNode` para facilitar a criação de múltiplas perguntas. Essa experiência aprimorou minhas habilidades em HTML, CSS e no desenvolvimento de aplicações web interativas.
